# Glossary

- **ATC**: Air Traffic Control.
- **Callsign**: Radio identifier for an aircraft (e.g., `AFR345`, `SPEEDBIRD 42`).
- **Intent**: Operational command meaning extracted from phraseology.
- **Slot**: Structured value inside an intent (e.g., altitude value, speed value).
- **Readback**: Pilot repetition of a clearance/instruction.
- **Amendment**: Correction or replacement of a prior instruction in the same utterance.
- **Fallback state**: Explicit parse status indicating unresolved parse conditions.
- **Deterministic parser**: Rule-based parser with explainable, repeatable behavior.
